# Landing Page Spec (Structure Only) — Flowbite UI

## Goal
Drive sign-ups and first campaign creation.

## Visual style
- Clean, modern, light mode
- Flowbite components with Tailwind
- Strong product UI screenshots/mockups (sheet + campaign + session)

## Page sections

### 1) Top navigation
- Flowbite Navbar
- Left: logo + product name
- Right: links (Product, How it works, Pricing, FAQ)
- CTA: Sign in / Create account

### 2) Hero
- Two-column layout (Flowbite grid)
- Left: headline + subheadline + two CTAs
  - Primary: Create a campaign
  - Secondary: View character sheet
- Right: product mock (character sheet UI)

### 3) “Campaign → Sessions” explanation
- Flowbite Cards (3 cards)
  - Campaign overview
  - Sessions (date/time, notes, attendance, XP)
  - Session assets (NPCs, maps, treasure)

### 4) Invite flow
- Stepper-like layout using Cards + icons
  1. Invite by email with roles
  2. Accept invite
  3. Join with character (mandatory)
- Callout highlighting “No member without character”

### 5) Character sheet highlight
- Tabs: Attributes / Skills / Powers / Spells / Inventory
- Screenshot or interactive demo placeholder
- Bullet list of sheet features (no copywriting, just feature placeholders)

### 6) Per-campaign progression
- Two-panel comparison:
  - Same character base, different campaign states (lvl 10 vs lvl 1)
- Simple diagram explaining “base vs campaign instance”

### 7) Social proof / credibility
- Placeholder testimonial cards
- Placeholder usage stats (optional)

### 8) FAQ
- Flowbite Accordion
- Topics:
  - Can I switch character? (No)
  - Can I join without a character? (No)
  - Will you support other systems? (Future)
  - Is it mobile-friendly? (Yes)

### 9) Final CTA
- Big Card or banner
- Primary CTA: Create account
- Secondary CTA: Sign in

### 10) Footer
- Links: Terms, Privacy, Contact
- Social links
