# Create Campaign Page / Modal

Route: /app/campaigns/new

## Objective
Create a new campaign.

---

## Form Fields

- Title (Input)
- Description (Textarea)
- Frequency (Select)
- Status (Select)

Buttons:
- Cancel
- Create Campaign

---

## Components

- Input
- Textarea
- Select
- Button
- Modal (optional implementation)

---

## Validation

- Title required
- Frequency required

---

## Success Flow

- Redirect to campaign detail page
