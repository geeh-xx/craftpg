# Create Session Page

Route: /app/campaigns/[id]/sessions/new

## Objective
Create a new session within a campaign.

---

## Form Fields

- Title
- Date/Time
- Summary
- Notes

---

## Tabs Inside Session

- Attendance
- NPCs
- Maps
- Treasure

---

## Attendance Tab

Table:
- Character
- Present (checkbox)
- XP gained (input)

---

## NPCs Tab

List:
- NPC name
- Notes
- Stat payload

---

## Maps Tab

List:
- Title
- Storage path
- Notes

---

## Treasure Tab

List:
- Title
- JSON payload

---

## Components

- Input
- Datetime picker
- Tabs
- Table
- Checkbox
- Modal
